1   package es.ull.etsii.Recommendations;
2   
3   import java.util.ArrayList;
4   
5   /**
6    * Implemente esta interfaz para permitir que su código se integre con nuestro sitio web.
7    *
8    * <p>Cuando los usuarios visitan por primera vez el sitio web de recomendación, nuestro código llamará al método <code>
9    * getItemsToRate()</code> para obtener una lista de películas para mostrar en la página web para que los usuarios las califiquen.
10   *
11   * <p>Cuando un usuario envía sus calificaciones, nuestro código llamará al método <code>
12   * printRecommendationsFor</code> para obtener recomendaciones basadas en las calificaciones de los usuarios. La identificación
13   * dado a este método es para un nuevo evaluador que ya hemos agregado a RaterDatabase con
14   * calificaciones de las películas devueltas por el primer método. Lo que sea que se imprima de ese método será
15   * mostrado en la página web: HTML, texto sin formato o información de depuración.
16   */
17  public interface Recommender {
18    /**
19     * Este método devuelve una lista de ID de películas que se usarán para buscar las películas en el
20     * MovieDatabase y preséntelos a los usuarios para que los califiquen.
21     *
22     * <p>Las películas devueltas en la lista se mostrarán en una página web, por lo que el número que elija
23     * puede afectar el tiempo de carga de la página y la disposición de los usuarios a calificar las películas. Para
24     * ejemplo, 10-20 debería estar bien, 50 o más sería demasiado.
25     *
26     * <p>No hay restricciones sobre el método que utiliza para generar esta lista de películas: la mayoría
27     * películas recientes, películas de un género específico, películas elegidas al azar o simplemente tu favorita
28     * películas.
29     *
30     * <p>Las clasificaciones de estas películas crearán el perfil de un nuevo evaluador que se usará para
31     * comparar con para encontrar recomendaciones.
32     */
33    ArrayList<String> getItemsToRate();
34  
35    /**
36     * Este método no devuelve nada, pero imprime una tabla HTML de las películas recomendadas para el
37     * evaluador dado.
38     *
39     * <p>El HTML impreso se mostrará en una página web, por lo que el número que elija mostrar puede
40     * afecta el tiempo que tarda la página en cargarse. Por ejemplo, es posible que desee limitar el número impreso
41     * solo a las 20-50 mejores películas recomendadas o a las películas no calificadas por el calificador dado.
42     *
43     * <p>También puede incluir estilo CSS para su tabla utilizando el &lt;estilo&gt; etiqueta antes de imprimir
44     * la mesa. No hay restricciones sobre qué películas imprime, en qué orden las imprime,
45     * o qué información incluyes sobre cada película.
46     *
47     * @param webRaterID el ID de un nuevo evaluador que ya se ha agregado a RaterDatabase con
48     * calificaciones de las películas devueltas por el método getItemsToRate
49     */
50    void printRecommendationsFor(String webRaterID);
51  }
