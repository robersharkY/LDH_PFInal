\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_range_resource}{Range\+Resource}}} class represents a range of integer (or whole) numbers and allows access to these numbers using the method {\ttfamily sequence}. The integers can then be iterated over using a {\ttfamily for} loop

Example usage\+:


\begin{DoxyPre}
// this prints the square number sequence for the first 9 numbers:
//   1, 4, 9, 16, 25, 36, 49, 64, 81 
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_range_resource}{RangeResource}} rr = new RangeResource(1, 10);
for (int value : rr.sequence()) \{
    System.out.println(value * value);
\}
\end{DoxyPre}


Another example usage\+:


\begin{DoxyPre}
// this prints the odd numbers in decreasing order within the given range:
//   37, 35, 33, 31, 29, 27, 25, 23
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_range_resource}{RangeResource}} rr = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_range_resource}{RangeResource}}(37, 21, -\/2);
for (int value : rr.sequence()) \{
    System.out.println(value);
\}
\end{DoxyPre}
