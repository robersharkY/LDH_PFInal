\doxysubsubsection*{Defining column names}

To define the column names you want to use to access records, write\+: 


\begin{DoxyPre}
CSVFormat.EXCEL.withHeader("{}Col1"{}, "{}Col2"{}, "{}Col3"{});
\end{DoxyPre}


Calling {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_format_afe402eb4440c642f7f81fd6d345dedc0}{with\+Header(\+String...)}}} let\textquotesingle{}s you use the given names to address values in a {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_record}{es.\+apachecsv.\+org.\+apache.\+commons.\+csv.\+CSVRecord}}}, and assumes that your CSV source does not contain a first record that also defines column names.

If it does, then you are overriding this metadata with your names and you should skip the first record by calling {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_format_adf81a75b2e1fe65469afaa32cacbb3e7}{with\+Skip\+Header\+Record(boolean)}}} with {\ttfamily true}. 

\doxysubsubsection*{Parsing}

You can use a format directly to parse a reader. For example, to parse an Excel file with columns header, write\+: 


\begin{DoxyPre}
Reader in = ...;
CSVFormat.EXCEL.withHeader("{}Col1"{}, "{}Col2"{}, "{}Col3"{}).parse(in);
\end{DoxyPre}


For other input types, like resources, files, and URLs, use the static methods on {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_parser}{es.\+apachecsv.\+org.\+apache.\+commons.\+csv.\+CSVParser}}}. 

\doxysubsubsection*{Referencing columns safely}

If your source contains a header record, you can simplify your code and safely reference columns, by using {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_format_afe402eb4440c642f7f81fd6d345dedc0}{with\+Header(\+String...)}}} with no arguments\+: 


\begin{DoxyPre}
CSVFormat.EXCEL.withHeader();
\end{DoxyPre}


This causes the parser to read the first record and use its values as column names.

Then, call one of the {\ttfamily \mbox{\hyperlink{classes_1_1apachecsv_1_1org_1_1apache_1_1commons_1_1csv_1_1_c_s_v_record}{es.\+apachecsv.\+org.\+apache.\+commons.\+csv.\+CSVRecord}}} get method that takes a String column name argument\+: 


\begin{DoxyPre}
String value = record.get("{}Col1"{});
\end{DoxyPre}


This makes your code impervious to changes in column order in the CSV file.
