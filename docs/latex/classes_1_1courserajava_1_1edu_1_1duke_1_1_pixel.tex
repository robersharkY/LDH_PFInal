\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_pixel}{Pixel}}} class represents a color as its component values of red, green, blue, as well as alpha (for transparency).

Each of the component values of a pixel must have a value between 0 and 255. If a value is given outside that range, it is changed to be within that range. As such, a negative value would be set to 0 and a value greater than 255 would be set to 255.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_image_resource}{ImageResource}} image = new ImageResource();
for (\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_pixel}{Pixel}} p : image.pixels()) \{
    int red = p.getRed();
    int green = p.getGreen();
    int blue = p.getBlue();
    int average = (red + green + blue) / 3;
    p.setRed(average);
    p.setGreen(average);
    p.setBlue(average);
\}
image.draw();
\end{DoxyPre}
