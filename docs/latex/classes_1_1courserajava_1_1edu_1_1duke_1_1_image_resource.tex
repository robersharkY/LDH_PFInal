\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_image_resource}{Image\+Resource}}} class represents an image as a grid of {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_pixel}{Pixel}}} objects and allows access to all of them, using the method {\ttfamily pixels}. These pixels can then be iterated over using a {\ttfamily for} loop.

Note, no changes made to the pixels in this image affect the original image opened unless you use the method {\ttfamily save}.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_image_resource}{ImageResource}} image = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_image_resource_a14f71a8e0788f3ea02141284dc60ae6d}{ImageResource()}};
for (\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_pixel}{Pixel}} p : image.pixels()) \{
    int red = p.getRed();
    int green = p.getGreen();
    int blue = p.getBlue();
    int average = (red + green + blue) / 3;
    p.setRed(average);
    p.setGreen(average);
    p.setBlue(average);
\}
image.draw();
\end{DoxyPre}
