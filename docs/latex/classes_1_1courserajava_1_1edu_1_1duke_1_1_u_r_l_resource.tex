\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_u_r_l_resource}{URLResource}}} class opens a connection to a URL and allows access to the contents of the web page a line at a time, using the method {\ttfamily lines}, or a word at a time, using the method {\ttfamily words}. These strings can then be iterated over using a {\ttfamily for} loop.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_u_r_l_resource}{URLResource}} ur = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_u_r_l_resource}{URLResource}}("{}http://www.dukelearntoprogram.com/"{});
for (String s : ur.lines()) \{
    // print or process s
\}
\end{DoxyPre}


If each line of the web page represents separated data values, because its a CSV file, then the user can get a {\ttfamily get\+CSVParser} object to access that data more directly, using one of the {\ttfamily get\+CSVParser} methods.

Example CSV usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_u_r_l_resource}{URLResource}} ur = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_u_r_l_resource}{URLResource}}("{}http://www.dukelearntoprogram.com/course2/java/food.csv"{});
for (CSVRecord record : ur.getCSVParser()) \{
    // print or process fields in record
    String name = record.get("{}Name"{});
    // other processing
\}
\end{DoxyPre}
