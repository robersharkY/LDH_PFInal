\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_file_resource}{File\+Resource}}} class represents a file and allows access to its contents a line at a time, using the method {\ttfamily lines}, or a word at a time, using the method {\ttfamily words}. These strings can then be iterated over using a {\ttfamily for} loop.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_file_resource}{FileResource}} fr = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_file_resource_a4e9a7561227b2831a49301ea8837e85d}{FileResource()}};
for (String s : fr.words()) \{
    // print or process s
\}
\end{DoxyPre}


If each line of the file represents separated data values, because its a CSV file, then the user can get a {\ttfamily get\+CSVParser} object to access that data more directly, using one of the {\ttfamily get\+CSVParser} methods.

Example CSV usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_file_resource}{FileResource}} fr = new \mbox{\hyperlink{classes_1_1courserajava_1_1edu_1_1duke_1_1_file_resource}{FileResource}}("{}food.csv"{});
for (CSVRecord record : fr.getCSVParser()) \{
    // print or process fields in record
    String name = record.get("{}Name"{});
    // other processing
\}
\end{DoxyPre}
